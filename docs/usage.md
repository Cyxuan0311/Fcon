# 使用指南

## 快速开始

### 1. 初始化磁盘

1. 打开操作面板的"文件系统配置"标签页
2. 选择文件系统类型（FAT32/Ext4/NTFS）
3. 设置总块数和块大小
4. 点击"初始化磁盘"按钮

### 2. 创建文件

1. 切换到"文件操作"标签页
2. 输入文件名和文件大小（KB）
3. 选择分配算法（连续/链接/索引）
4. 点击"创建文件"按钮

### 3. 查看可视化

- **3D磁盘视图**: 中间区域显示磁盘块的3D可视化
- **目录树**: 在"文件目录"标签页查看文件结构
- **性能数据**: 右侧数据面板显示实时监控和图表

## 功能详解

### 文件系统配置

#### 导入JSON文件
支持导入预配置的JSON文件：
1. 点击"上传JSON文件"按钮
2. 选择JSON文件
3. 系统自动解析并应用配置

JSON格式示例：
```json
{
  "fileSystemType": "FAT32",
  "disk": {
    "totalBlocks": 1000,
    "blockSize": 4096,
    "files": [...]
  }
}
```

#### 磁盘参数
- **总块数**: 磁盘总块数（100-10000）
- **块大小**: 每个块的大小，单位KB（1-64）

### 文件操作

#### 创建文件
- **文件名**: 必填，不能与同目录下文件重名
- **文件大小**: 必填，单位KB，最小1KB
- **分配算法**: 选择文件分配算法

#### 删除文件
1. 在文件目录中点击选择文件
2. 点击"删除选中文件"按钮
3. 确认删除操作

#### 创建目录
1. 切换到"文件目录"标签页
2. 点击"新建目录"按钮
3. 输入目录名称
4. 确认创建

### 目录导航

#### 切换目录
- **下拉选择**: 在"文件操作"标签页使用下拉菜单
- **目录树**: 在"文件目录"标签页点击目录进入
- **路径导航**: 点击路径中的目录名快速跳转

#### 目录操作
- **返回上级**: 点击"返回上级"按钮
- **返回根目录**: 点击"根目录"按钮

### 碎片整理

1. 在操作面板底部点击"执行碎片整理"按钮
2. 确认操作
3. 系统自动整理碎片，移动磁盘块到连续区域
4. 查看整理结果（移动块数、碎片率变化）

### 终端命令

打开终端窗口，支持以下命令：

#### 文件操作
- `ls` / `dir`: 列出当前目录文件
- `cd <目录>`: 切换目录
- `pwd`: 显示当前路径
- `mkdir <目录名>`: 创建目录
- `touch <文件名> [大小KB]`: 创建文件
- `rm <文件名>`: 删除文件

#### 信息查看
- `cat <文件名>`: 查看文件信息
- `stat <文件名>`: 查看详细信息
- `find <名称>`: 查找文件
- `tree [目录]`: 显示目录树
- `du [目录]`: 显示目录大小
- `df`: 显示磁盘使用情况
- `info`: 显示系统信息

#### 其他
- `history [数量]`: 显示命令历史
- `echo <文本>`: 输出文本
- `clear` / `cls`: 清空终端
- `exit` / `quit`: 退出终端
- `help`: 显示帮助信息

### 性能监控

#### 实时指标
- **磁盘利用率**: 已使用块数 / 总块数
- **碎片率**: 非连续块占比
- **IO吞吐量**: IO操作速度
- **平均响应时间**: 操作平均耗时

#### 数据图表
- **柱状图**: 不同操作类型的耗时对比
- **折线图**: IO吞吐量随时间的变化趋势
- **饼图**: 磁盘空间使用占比

#### 数据导出
1. 在数据面板底部点击"导出数据"按钮
2. 系统生成Excel文件
3. 文件包含所有性能数据记录

### 3D交互

#### 鼠标控制
- **左键拖拽**: 旋转场景
- **滚轮**: 缩放场景
- **右键拖拽**: 平移场景

#### 选择操作
- **点击磁盘块**: 选中对应的文件
- **点击文件节点**: 选中文件并高亮显示
- **双击目录**: 进入目录

## 快捷键

- `Ctrl + N`: 创建文件（需在文件操作标签页）
- `Delete`: 删除选中文件
- `Ctrl + R`: 执行碎片整理
- `ESC`: 关闭终端窗口
