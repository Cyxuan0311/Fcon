# 功能特性

## 核心功能

### 1. 文件系统可视化

#### 磁盘可视化
- **3D网格展示**: 使用Three.js渲染磁盘块的3D网格
- **状态区分**: 不同颜色区分空闲块、已使用块、碎片块
- **动态更新**: 实时更新磁盘块状态和颜色
- **交互选择**: 点击选中磁盘块，查看详细信息

#### 目录树可视化
- **3D层级树**: 以3D树状结构展示文件系统目录层次
- **节点表示**: 球体节点表示文件和目录
- **连接线**: 可视化展示父子目录关系
- **颜色编码**: 每个文件/目录使用唯一颜色标识

### 2. 文件系统操作

#### 文件管理
- **创建文件**: 支持指定文件名、大小、分配算法
- **删除文件**: 删除文件并自动释放磁盘块
- **创建目录**: 创建目录结构
- **目录导航**: 支持目录切换和导航

#### 分配算法
支持三种磁盘分配算法：
- **连续分配**: 查找连续的空闲块序列
- **链接分配**: 随机分配空闲块，通过链表连接
- **索引分配**: 使用索引块管理数据块

#### 碎片整理
- **紧凑式整理**: 将分散的磁盘块移动到连续区域
- **自动更新**: 自动更新文件块映射关系
- **性能统计**: 显示整理结果（移动块数、碎片率变化）

### 3. 性能监控

#### 实时监控
- **磁盘利用率**: 实时显示磁盘使用百分比
- **碎片率**: 计算并显示磁盘碎片率
- **IO吞吐量**: 统计IO操作吞吐量
- **响应时间**: 记录平均响应时间

#### 数据图表
- **柱状图**: 操作耗时对比分析
- **折线图**: IO吞吐量变化趋势
- **饼图**: 磁盘空间占比可视化

#### 数据导出
- **Excel导出**: 支持将性能数据导出为Excel格式
- **数据格式化**: 自动格式化数据（中文列名、时间戳）

### 4. 交互功能

#### 操作面板
- **文件系统配置**: 配置文件系统类型（FAT32/Ext4/NTFS）、总块数、块大小
- **磁盘初始化**: 初始化磁盘参数
- **文件操作**: 创建/删除文件，支持参数验证
- **目录管理**: 目录创建、切换、导航

#### 3D交互
- **鼠标控制**: 支持鼠标旋转、缩放、平移3D场景
- **点击选择**: 点击文件节点或磁盘块进行选择
- **高亮显示**: 选中项高亮显示

#### 终端模拟
- **命令行界面**: 提供类Unix终端界面
- **文件系统命令**: 支持ls、cd、mkdir、touch、rm等命令
- **命令历史**: 支持命令历史记录和导航

### 5. 教学引导

#### 引导内容
- **7步引导流程**: 完整的操作引导
- **功能说明**: 详细的功能介绍
- **算法原理**: 解释文件系统算法原理
- **操作指导**: 步骤化的操作说明

#### 交互体验
- **美观弹窗**: 现代化的引导界面
- **进度显示**: 显示当前步骤和总步骤
- **导航控制**: 上一步/下一步/跳过功能
- **状态保存**: 引导完成状态保存到LocalStorage
